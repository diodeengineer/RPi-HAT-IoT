### RPi-HAT-IoT
A custom Raspberry Pi HAT designed for high-performance IoT applications. This board simplifies the integration of sensors and connectivity, turning a standard Raspberry Pi into a robust IoT Gateway or Edge device.

### Overview
This project provides a hardware-level expansion for the Raspberry Pi 40-pin header. It is designed to bridge the gap between physical sensors and cloud-based IoT platforms.

### Features
Form Factor: Standard Raspberry Pi HAT (compatible with Pi 3, 4, and 5).

Sensor Suite: (e.g., Temperature, Humidity, and Pressure sensors).

Indicator: Status LEDs for power and data transmission.
